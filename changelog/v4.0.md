# **What's changed?**

## [4.1.4](https://github.com/nishkohli96/eslint-config/tree/v4.1.4)

**Released - 1 April, 2026**

### Fixes 🔧
- Increase [@stylistic/max-len](https://eslint.style/rules/max-len) from **80** to **120**
- Add package entry points (`main`, `exports["."]`, and root `types` on **eslint-flat-config**) so both published configs resolve to the JS preset by default; bundle-size and other tools that need a package entry no longer fail when only subpath exports existed
